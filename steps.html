Reproducibility & Instructions


Code Access: https://github.com/BVSSUMANTH/Mutilmodel_gener_classification

Setup Guide:

  1. It is recommended to create a virtual environment first, use the command "python -m venv name_of_env" and then activate the environment with the command "name_of_env\Scripts\activate"

  2. Install necessary packages: "!pip install requests pandas torch torchvision transformers" or you can run the first cell of EDA file to install these.

  3. Make sure you have downloaded glove.6B.zip and extracted the files into the directory you are working. This is step is essential for the text model.

Data Downloading:

  1. After following all the setup guide steps simply open the EDA.ipynb file and start running the cells one by one.

  2. After running the cell with "def fetch_movies_for_year(api_key, year):" function, the data will automatically save in a CSV file in the working directory.

    Note: make sure you place your TMDB api key before running the code.

Run Commands or Notebook Usage:

  1. Once you have downloaded the data you are all set for the next steps

  2. First run the EDA file completely and then you can run the text model or image model and fusion model

  3. In the text model you will find "glove_path", initially it is set to use "glove.6b.100d.txt" but the zip file has four different types of glove embeddings. If you would like change the type of embedding used you change the path of the file.
